### 1. 数据分离与异步加载技术

#### 核心实现
```javascript
// 异步加载数据
async function loadData() {
    try {
        // 检查本地缓存
        const cachedData = localStorage.getItem('cdut_spiral_data');
        const cacheTimestamp = localStorage.getItem('cdut_spiral_data_timestamp');
        const cacheValid = cacheTimestamp && 
            (Date.now() - parseInt(cacheTimestamp)) < 24 * 60 * 60 * 1000;
        
        if (cachedData && cacheValid) {
            data = JSON.parse(cachedData);
            initVisualization();
            return;
        }
        
        // 从服务器加载
        const response = await fetch('cdut_spiral_data.json');
        data = await response.json();
        
        // 缓存数据
        localStorage.setItem('cdut_spiral_data', JSON.stringify(data));
        localStorage.setItem('cdut_spiral_data_timestamp', Date.now().toString());
        
        initVisualization();
    } catch (error) {
        console.error('数据加载失败:', error);
    }
}
```

#### 技术要点
- ✅ **Fetch API**：实现异步数据加载
- ✅ **localStorage**：实现本地缓存
- ✅ **时间戳验证**：24小时缓存有效期
- ✅ **错误处理**：完善的异常处理机制

### 2. 3D螺旋塔可视化技术

#### 核心算法
```javascript
// 3D旋转投影算法
function project3D(x, y, z, cameraX, cameraY, cameraZ, cameraRotation) {
    // 旋转
    const rotatedX = x * Math.cos(cameraRotation) - z * Math.sin(cameraRotation);
    const rotatedZ = x * Math.sin(cameraRotation) + z * Math.cos(cameraRotation);
    
    // 透视投影
    const scale = 1000 / (rotatedZ + cameraZ);
    const screenX = rotatedX * scale + canvas.width / 2;
    const screenY = y * scale + canvas.height / 2;
    
    return { x: screenX, y: screenY, scale: scale };
}
```

#### 技术要点
- ✅ **Canvas 2D**：高性能绘图
- ✅ **3D旋转算法**：模拟3D效果
- ✅ **透视投影**：增强立体感
- ✅ **分层渲染**：按深度排序绘制

### 4. 数据处理技术

#### Python数据处理脚本
```python
import pandas as pd
import json

# 读取Excel数据
df = pd.read_excel('本科专业发展-导出数据(1).xlsx')

# 数据清洗和转换
data = []
for year in df['年份'].unique():
    year_data = df[df['年份'] == year]
    departments = []
    
    for dept in year_data['院系'].unique():
        dept_data = year_data[year_data['院系'] == dept]
        majors = []
        
        for _, row in dept_data.iterrows():
            majors.append({
                'name': row['专业名称'],
                'code': row['专业代码'],
                'degree': row['学位'],
                'duration': row['学制']
            })
        
        departments.append({
            'name': dept,
            'majors': majors,
            'majorCount': len(majors)
        })
    
    data.append({
        'year': year,
        'departments': departments,
        'majorCount': len(df[df['年份'] == year])
    })

# 保存为JSON
with open('cdut_spiral_data.json', 'w', encoding='utf-8') as f:
    json.dump(data, f, ensure_ascii=False, indent=2)
```

#### 技术要点
- ✅ **Pandas**：Excel数据读取和处理
- ✅ **数据清洗**：去重、格式转换
- ✅ **JSON序列化**：结构化数据输出
- ✅ **批量处理**：自动化数据转换

## 🎯 技术选型理由

### 为什么选择Canvas而不是WebGL？
- ✅ **兼容性**：Canvas支持所有现代浏览器
- ✅ **开发成本**：Canvas API更简单易学
- ✅ **性能**：对于本项目的复杂度，Canvas足够高效
- ✅ **文件大小**：无需引入额外的3D库

### 为什么选择JSON而不是数据库？
- ✅ **轻量级**：JSON文件体积小
- ✅ **易部署**：无需服务器端数据库
- ✅ **易维护**：可以直接编辑JSON文件
- ✅ **高性能**：本地加载速度快

### 为什么选择localStorage而不是IndexedDB？
- ✅ **简单易用**：API更简洁
- ✅ **足够用**：本项目数据量较小
- ✅ **兼容性**：支持所有现代浏览器
- ✅ **性能**：读写速度快
